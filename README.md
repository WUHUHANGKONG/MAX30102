# Waffle Nano老人用便携血氧心率测量仪

#### 介绍
制作一个通过MAX30102脉搏血氧浓度传感器帮助老年人群体随时监测自己的健康状况，持续读取red，ir的值,通过算法计算得到血氧饱和度以及心率，给用户较大字体显示血氧饱和度以及心率图像，以便于用于居家老年人的血氧测量，通过蜂鸣器来警示血氧以及心率等数值的异常，并上传到thingswise idos平台实现数据可视化，实现长期监控。
